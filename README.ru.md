# 🐱 Meow Magic

Meow Magic это простой и увлекательный веб-сайт, который генерирует случайные изображения кошек одним нажатием кнопки.

![MeowMagicDemo](https://github.com/KrishBharadwaj5678/Meow-Magic/raw/main/MeowMagic.png)

## ✨ Возможности

| Возможность                    | Описание                                                               |
| ------------------------------ | ---------------------------------------------------------------------- |
| 🐾 Случайные изображения кошек | Мгновенно загружает новые изображения кошек при каждом нажатии         |
| 🎨 Простой и чистый интерфейс  | Разработан для удобного использования без лишних отвлекающих элементов |
| ⚡ Быстрый и лёгкий             | Минимальное время загрузки и оптимизированная производительность       |

---

## ⚙️ Технологический стек

| Технология        | Назначение                                                        |
| ----------------- | ----------------------------------------------------------------- |
| 🌐 **HTML**       | Формирует структуру содержимого веб-страницы                      |
| 🎨 **CSS**        | Добавляет стили для привлекательного пользовательского интерфейса |
| ⚙️ **JavaScript** | Получает и отображает случайные изображения кошек                 |
