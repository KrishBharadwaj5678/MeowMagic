# 🐱 Meow Magic

Meow Magic は、ワンクリックでランダムな猫の画像を生成する、シンプルで楽しいウェブサイトです。

![MeowMagicDemo](https://github.com/KrishBharadwaj5678/Meow-Magic/raw/main/MeowMagic.png)

## ✨ 特徴

| 機能              | 説明                       |
| --------------- | ------------------------ |
| 🐾 ランダムな猫画像     | クリックするたびに新しい猫の画像を即座に取得   |
| 🎨 シンプルでクリーンなUI | スムーズで邪魔のないユーザー体験を実現      |
| ⚡ 高速・軽量         | 最小限の読み込み時間と最適化されたパフォーマンス |

---

## ⚙️ 技術スタック

| 技術                | 用途                |
| ----------------- | ----------------- |
| 🌐 **HTML**       | ウェブページの構造を作成      |
| 🎨 **CSS**        | 視覚的に魅力的なUIのスタイリング |
| ⚙️ **JavaScript** | ランダムな猫画像を取得して表示   |
