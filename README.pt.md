# 🐱 Meow Magic

O Meow Magic é um site simples e divertido que gera imagens aleatórias de gatos com apenas um clique.

![MeowMagicDemo](https://github.com/KrishBharadwaj5678/Meow-Magic/raw/main/MeowMagic.png)

## ✨ Funcionalidades

| Funcionalidade                 | Descrição                                                   |
| ------------------------------ | ----------------------------------------------------------- |
| 🐾 Imagens Aleatórias de Gatos | Obtém novas imagens de gatos instantaneamente a cada clique |
| 🎨 Interface Simples e Limpa   | Projetada para uma experiência suave e sem distrações       |
| ⚡ Rápido e Leve                | Tempo de carregamento mínimo e desempenho otimizado         |

---

## ⚙️ Tecnologias Utilizadas

| Tecnologia        | Finalidade                                               |
| ----------------- | -------------------------------------------------------- |
| 🌐 **HTML**       | Estrutura o conteúdo da página web                       |
| 🎨 **CSS**        | Adiciona estilos para uma interface visualmente atraente |
| ⚙️ **JavaScript** | Busca e exibe imagens aleatórias de gatos                |
